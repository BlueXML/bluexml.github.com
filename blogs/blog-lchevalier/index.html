Génération de la documentation de l'application avec SIDE

Date: Mon Dec 10 00:00:00 +0100 2012

Une documentation de chaque modèle de donnée S-IDE peut être générée automatiquement avec S-IDE, pour cela il faut cocher la case "Documentation" de l'onglet génération. Format Le format utilisé est l'Open Document, notamment utilisé par OpenOffice. Un fichier lisible par Open Office est un fichier jar (nommé en .odt) contenant : /META-INF/manifest.mf /META-INF/manifest.xml content.xml meta.xml mimetype settings.xml styles.xml Génération La génération se fait via un générateur spécifique par méta-modèle (Class, Form, Workflow, View on en déjà un par exemple) utilisant tous une base commune (le projet com.bluexml.side.Util.generator.documentation). Ce dernier permet de générer les fichiers 'fixes' : meta.xml, mimetype, setting.xml, styles.xml....

Date: Wed Sep 09 00:00:00 +0200 2009
